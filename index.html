Введение

JavaScript — это язык программирования, который используют для написания frontend- и backend-частей сайтов, а также мобильных приложений. Часто в текстах и обучающих материалах название языка сокращают до JS. Это язык программирования высокого уровня, то есть код на нем понятный и хорошо читается.

JS поддерживают все популярные браузеры. Во frontend-части сайтов язык используют для создания интерактива (анимаций, всплывающих форм, автозаполнения), так как он связан с HTML и CSS и может ими манипулировать. В backend-части с языком JavaScript работают на платформе Node.js. С ее помощью, например, разрабатывают серверные веб-приложения и подключают библиотеки. В поисковике Google на JavaScript работает строка автозаполнения, а Netflix, Uber, eBay используют его в своем backend.

JavaScript называют языком скриптов или сценариев. Скрипты — это набор инструкций, которые выполняются при загрузке страницы. Браузер самостоятельно интерпретирует код на JavaScript, для этого даже не требуется компиляция (перевод языка программирования в машинный код).

О JS

Для чего нужен JavaScript

В веб-приложениях HTML отвечает за разметку страницы. Расположение контента, внешний вид интерфейса описывается с помощью CSS, а JavaScript отвечает за интерактивность и функционал этого интерфейса. Вот примеры задач, которые решает JS:

  • Взаимодействие с пользователем. Всплывающие окна с полем для ввода почты или других данных, уведомления об использовании cookie.
  • Обработка данных HTML. JS помогает заполнить необходимые поля в формах, проверять буквенные и числовые значения, а также управлять HTML-элементами на странице. Например, в конструкторе лендингов Tilda можно перетаскивать блоки и по клику менять шрифты, цвета текста, фонов и другие значения.
  • Анимация. Это не только движущиеся объекты на странице и кнопки, но и карусель в галерее или плавная прокрутка страницы. Например, Pinterest с помощью JavaScript создал эффект бесконечной ленты.
  • Математические вычисления. С помощью JS-скриптов на сайтах реализованы калькуляторы, например подсчет суммы заказа в корзине интернет-магазина.

Что не получится сделать на JavaScript

Вот какие ограничения есть у JS:

  • JS не умеет читать, записывать или запускать файлы напрямую с устройства.
  • Чтобы связать всплывающее окно с основной страницей, придется писать дополнительный код, иначе они не смогут обмениваться введенными данными.
  • JS взаимодействует только с тем сервером, с которого открыта страница. Для работы со сторонними сайтами и сервисами придется прописать дополнительный код с обеих сторон, чтобы они могли взаимодействовать друг с другом.

Некоторые возможности

Приветствие

Как тебя зовут?

  1. Получаем доступ к элементу <input></input> с идентификатором "user name" и сразу же присваиваем ешл значение в переменную name

    var name = document.getElementById("user_name").value;

  2. Проверяем введённое пользователем значение, записанное в name. Поскольку <input></input> был с текстовым типом (атрибутом type="text"), мы берём длину строки. Если длина строки оказалась нулевой, то выводим сообщение о приветствии пользователя, без указания имени пользователя (alert("Привет друг!")), иначе же, если длина не нулевая, значит пользователь ввёл имя. Сохраняем его в localStorage (локальное хранилище), указывая имя переменной "user_name" в которую запишется значение из переменной name (localStorage("user_name", name)). После чего выводим сообщение о приветствии с именем пользователя из переменной name (alert("Привет " + name + "!"))

    if (name.length == 0) {
	alert("Привет друг!");
} else {
	localStorage.setItem("user_name", name);
	alert("Привет " + name + "!");
}

Минимакс

Укажи значение 5-ти элементов и узнай минимальное и максимальное из них:

  1. Для того, чтобы наш Минимакс мог сравнивать и числа и строки. Напишем свою функцию подсчёта максимального и минимального значения.

    Создаём переменные: mass для хранения доступа ко всем необходимым элементам <input></input>, min и max для подсчёта минимального и максимального значения, min_index и max_index для получения индексов минимального и максимального значения, а также count для подсчёта текущего элемента

    var 
	mass = new Array(),
	min, 
	max, 
	min_index, 
	max_index, 
	count = 0;

  2. Получаем доступ ко всем нужным элементам с идентификаторами "mass_element_i". Переменная i выступает в роли счётчика цикла.

    for (var i = 0; i < 5; i++) {
	mass.push(document.getElementById("mass_element_" + i).value);
}

  3. Проводим проверку на количество введённых элементов. Если всё в порядке, то получаем значение нулевого элемента из mass и присваиваем его в min и max для дальнейшего подсчёта. Также устанавливаем min_index и max_index в индекс, к которому заранее прибавили еденицу для лучшей визуализации в сообщении с результатом

    if (mass.length == 0) {
	alert("Элементов не обнаружено!")
} else {
	if (isNaN(parseInt(mass[0]))) {
		min = mass[0].length;
		max = min;
	} else {
		min = parseInt(mass[0]);
		max = min;
	}
	min_index = 1;
	max_index = 1;
}

  4. Далее в цикле проходимся по всем элементам mass, начиная с 1-го, потому что нулевой мы уже обработали. Выясняем тип элемента: если значение элемента не число (isNaN(parseInt(mass[i]))), то вызываем свойство length и получаем длину строки и присваиваем в счётчик (count = mass[i].length), в противном случаем просто кладём в счётчик значение элемента (count = mass[i]).

    Затем, в том же цикле, выполняем две проверки: если значение счётчика оказалось меньше чем min, то в min присваиваем значение счётчика (count) а в min_index индекс текущего элемента с прибавленной единичкой (для лучше понятного отображения пользователю); аналогичную проверку выполняем для max

    for (var i = 1; i < mass.length; i++) {
	if (isNaN(parseInt(mass[i]))) {
		count = mass[i].length;
	} else {
		count = mass[i];
	}

	if (count < min) {
		min = count;
		min_index = i + 1;
	}
	if (count > max) {
		max = count;
		max_index = i + 1;
	}
}

  5. Выводим сообщение с результатом пользователю

    alert("Минимальный элемент " + min_index + ", максимальный элемент " + max_index);

Таймер

Засеки время с помощью таймера:

00:00:00

  1. Создём переменные: t для управления методом setInterval, вызывающим функцию через определённый промежуток времени, hours, minutes и seconds для подсчёта часов, минут и секунд соответственно, screen для получения доступа к элементу отображения таймера (<h3 id="timer_screen"><time></time></h3>) и buffer для обновления значения screen

    var 
	t,
	hours = 0,
	minutes = 0,
	seconds = 0,
	screen = document.getElementById("timer_screen"),
	buffer = "";

  2. Создаём функцию timer_count(), которая будет вести подсчёт часов, минут и секунд

    Сразу же увеличиваем значение seconds на 1. Далее проверям: если seconds дошёл до 60, то обновляем значение seconds, присваивая ему 0, и увеличиваем minutes на 1. Тут же проверяем minutes, как и seconds: если minutes дошёл до 60, сбрасываем его до нуля и увеличиваем hours на 1

    function timer_count() {
	seconds++;
	if (seconds >= 60) {
		seconds = 0;
		minutes++;
		if (minutes >= 60) {
			minutes = 0;
			hours++;
		}
	}
}

  3. Создаём функцию timer(), запускающую timer_count() с заданной периодичностью.

    Присваиваем в переменную t метод setTimeout(), которому передаём функцию timer_screen(), которую он будет запускать, и значение периода (1000 = 1 секунда), через который будет повторно вызвана функция, которую мы передали.

    function timer() {
	t = setTimeout(timer_screen(), 1000);
}

  4. Создаём функцию timer_screen(), отвечающую за изменение и корректное отображение элемента screen

    Сразу же запускаем функцию timer_count() для подсчёта часов, минут и секунд. Затем в buffer записываем отображение часов, минут и секунд, используя тернарные операторы (hours > 9 ? hours : "0" + hours). Присваиваем в screen через свойство textContext значение buffer и запускаем функцию timer() для повтороного вызова

    function timer_screen() {
	timer_count();
	buffer = hours > 9 ? hours : "0" + hours + ":";
	buffer += minutes > 9 ? minutes : "0" + minutes + ":";
	buffer += seconds > 9 ? seconds : "0" + seconds;
	screen.textContent = buffer;
	timer();
}

  5. Создаём ещё одну функцию timer_pause(), отвечающую за паузу таймера

    С помощью метода clearTimeout() останавливаем созданный Timeout t, передав его в метод в качетсве аргумента

    function timer_pause() {
	clearTimeout(t);
}

  6. Атрибут onclick на любом элементе html содержит информацию о действии при клике на этот элемент

    Добавим атрибуты onclick к кнопкам с идентификаторами "timer_on" и "timer_off". Таким образом, мы добавили действие при клике на кнопки, передав нужные функции: timer() и timer_pause() соответственно

    <button id="timer_on" onclick="timer()">Запусти!</button>
<button id="timer_off" onclick="timer_pause()">Пауза</button>

  4. Создаём переменную рекурсивную функцию show(value), принимающую предыдущее значение прозрачности (value)

    Проверяем: если value меньше единицы (100%-ой прозрачности), прибавляем к value 0.001 и присваиваем его нашему красивому элементу, обратившись к стилям (super_screen.style.opacity = value). Затем, воспользовавшись методом setTimeout() запускаем show(value) со значением value+0.001 и периодом 50мс

    function show(value) {
	if (value < 1) {
		value += 0.001;
		super_screen.style.opacity = value;
		setTimeout(show(value + 0.001), 50);
	}	
}

  5. Создаём переменную рекурсивную функцию hide(value), принимающую предыдущее значение прозрачности (value)

    По аналогии с show(value) убавляем прозрачность до 0. При достижении 0 скрываем красивый элемент (super_screen.style.display = "none")

    function hide(value) {
	if (value > 0) {
		value -= 0.001;
		super_screen.style.opacity = value;
		setTimeout(hide(value - 0.001), 50);
	} else {
		super_screen.style.display = "none";
	}
}

  6. Далее создаём функцию surprise(), отвечающую за появление красивого элемента

    Сразу же переключаем свойсто display у super_screen в значение "block", чтобы он стал видимым. Создаём все необходимые переменные: date_field и user_name_field для получения доступа к нужным элементам красивого элемента для их формирования. Также создаём свойство options с указанием форматов полей для принятия текущей даты

    function surprise() {
	super_screen.style.display = "block";
	
	var 
		date_field = document.getElementById("surprise_date")
		user_name_field = document.getElementById("surprise_user_name");

	var options = {
		year: 'numeric',
		month: 'numeric',
		day: 'numeric',
		timezone: 'UTC'
	};
	
	date_field.textContent = new Date().toLocaleString("ru", options);
	
	if (localStorage.getItem("user_name") != null) {
		var buff = "Хорошего дня, ";
		buff += localStorage.getItem("user_name") + "!";
		user_name_field.textContent = buff;
	} else {
		user_name_field.textContent = "Хорошего дня, друг!";
	}

	show(0.001);
}

    Присваиваем в date_field в свойство textContent (date_field.textContent) текстовое представление даты (new Date()), используя метод toLocaleString с указанием параметров "ru" и нашего свойства options

    Далее присваиваем в user_name_field в свойство textContent строку приветствия. Проверяя localStorage на наличие имени пользователя, введённом в Приветствии

    Запускаем функию show(value) со значением 0.001

  7. Для элемента <button id="super_button"></button> добавим атрибут onclick, в котором укажем нашу функцию surprise()

    <button id="super_button" onclick="surprise()">Нажми на меня!</button>
